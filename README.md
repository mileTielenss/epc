# EPC Plaatsbezoek

Mobile-first offline PWA voor EPC-plaatsbezoeken (Vlaanderen). Vanilla
HTML/CSS/JS, nul dependencies, geen build-stap.

**`SPEC.md` is de volledige en enige beschrijving van de app** — features,
datamodel, PDF-generator, failsafes, testen en release. Dit bestand is bewust
niet meer dan een verwijzing.

- Live: https://miletielenss.github.io/epc/
- Installeren op iPhone: open de URL in Safari → Deel → "Zet op beginscherm".
- Iconen opnieuw genereren: `node tools/make-icons.js`
